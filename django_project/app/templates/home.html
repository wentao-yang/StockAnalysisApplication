About this API

This API is used as part of a project that stores historical stock data into InfluxDB with Django as the backend framework. You can learn more about the responses of the API on the right (the box brackets indicate variables when closed on the same line), and the link to the GitHub repository of the project is below:

Link to GitHub Repo

GET Historical Prices for a Stock

URL: /api/symbol=[stock ticker symbol]

{
    'symbol': [stock ticker symbol]
    'data':
        {
            [time]: [stock price]
            ...
        }
}


GET All Stock Ticker Symbols in Database

URL: /api/symbols

{
    'names':
        [
            [stock ticker symbol]
            ...
        ]
}


Note: this API only works if the InfluxDB database is activated and properly set up.
